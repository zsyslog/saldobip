# SaldoBip!

Esta es un ejemplo de alternativa para consular el saldo de su tarjeta Bip!

Realizado con Nodejs (Cheerio + Express + Request + forever)

Para consultar el saldo:

```
http://host/saldo/:NUM_TARJETA/:RUT
```
